This is a template for a serverless site using AWS S3, Lambda, Cognito, IAM, and API Gateway.

The template is also powered by a CI/CD Pipeline through AWS CodePipeline, CodeBuild, and CloudFormation & SAM, as well as GitHub.

The site itself leverages Bootstrap and FontAwesome, and is generated through Jekyll.

The documentation (i.e. setup instructions, modification details) and full source code for this template as well as the pipeline that powers it is available within the GitHub repository.
